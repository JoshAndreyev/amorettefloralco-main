Basic Website Template (HTML/CSS/JS)
=====================================

Files
- index.html  → one-page site with Hero, Work, About, Contact
- styles.css  → clean, responsive CSS with variables
- script.js   → optional mobile-nav toggling
- assets/     → placeholder images and favicon

Quick Start
1) Open index.html in your browser to preview.
2) Replace placeholder images in /assets with your own.
3) Update contact info + Formspree endpoint in the form action.
4) Deploy to Vercel/Netlify or host on any static server.
